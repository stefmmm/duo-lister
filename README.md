# duo-lister
List's someones Duolingo profile in discord.
With the power of webhooks!

#### How it works:

1. Edit app.py
1. Change 'YOUR WEBHOOK URL' to well.. your webhook url.
1. Save file.
1. Run app.py in cmd or powershell
1. Enter a username.
1. Press Enter
1. Magic

# duo-lister
List's someones Duolingo profile in discord.
With the power of webhooks!

#### How it works:

1. Edit config.py
1. Change 'YOUR WEBHOOK ID' and 'YOUR WEBHOOK SECRET' this info can be found in the webhook url.
1. Save file.
1. Run app.py in cmd or powershell
1. Enter a username.
1. Press Enter
1. Magic
